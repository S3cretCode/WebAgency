# Premium WebServices & Co.

**Digital Infrastructure for Growing Enterprises.**

A premium corporate agency website built with Next.js 14, Tailwind CSS v3, and the App Router. Fully responsive, SEO-optimized, and Vercel-ready.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS v3
- **Fonts:** Inter + Playfair Display (via next/font)
- **Animations:** CSS-only with Intersection Observer

## Deployment
Push to GitHub and connect to [Vercel](https://vercel.com) — auto-detects Next.js.
